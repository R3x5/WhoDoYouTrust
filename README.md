# WhoDoYouTrust

This repository documents how authentication decisions are made inside Active Directory,
with a focus on trust boundaries, protocol semantics, and decision points rather than tools or exploitation.

The goal is to describe ***what the system does*** and ***why Kerberos accepts what it accepts***.
